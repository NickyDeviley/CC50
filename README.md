# Curso de ciência da computação de Harvard

Este repositório guarda os arquivos que fiz durante o curso CC50 de Harvard, disponibilizado pela Fundação Estudar através da sua iniciativa Estudar Fora.

Link para o curso: https://www.estudarfora.org.br/cc50-v2/

O curso e este repositorio estão separados em semanas;
Todo arquivo cujo nome começa com "P_" são testes que fiz por conta própria ao longo do curso.

Status: Em andamento - Aulas concluídas 13 de 90;

Semana 0 - Sistema Binário, tabela ascii, algoritmo busca binária, jogo em Scratch;
Semana 1 - Linguagem de programação C;
